# Élan Boutique Website

A premium, mobile‑first website for a local clothing boutique, built with vanilla HTML, CSS, JavaScript, and a lightweight Node.js backend.

## Setup

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Open `http://localhost:3000`

All design tokens, components, and pages follow the comprehensive design system documented in `docs/DESIGN.md`.# clothing-shop
